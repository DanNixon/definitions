NeXus 4.3

NeXus 4.3 is the current release stream. To report or view any current issues, please consult the NeXus code bug/feature request system

4.3.1 (released 7th March 2013)

4.3.1 is the next official release since 4.3.0 - for a summary of the new and changed features see the release notes. The distribution has been tested with the help of the NeXus community via release candidates - if you wish to be involved with the future development of NeXus, please join the nexus mailing list. If any issues have been reported with 4.3.1 they will be listed here.

The 4.3.1 code can be browsed directly or downloaded via one of the links below:

Source

  • Compressed tar file (nexus-4.3.1.tar.gz)
  • Linux RPM source distribution

Binary

  • Linux RPM binary distribution
  • Microsoft Windows Installer (32 bit)
  • Microsoft Windows Installer (64 bit)
  • Apple Mac OS X Installer
Other binary distributions will be added in due course - if you require a particular binary which is not listed, please create a ticket in the Issue Tracking System.

There is doxygen documentation for the NeXus API. See here for code changes since 4.3.0

NeXus 3.0

NeXus 3.0 is the previous official release; the code base is currently frozen as work has moved onto the 4.0 release. The known issues with 3.0.0 are:
  • Various small memory leaks (tickets 19, 31, 34 and 45)
  • Inability to read two dimensional character arrays (ticket 7)
These issues have been fixed in 4.0
